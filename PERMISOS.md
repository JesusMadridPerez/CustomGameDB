# Permisos

Para esta aplicación seria nesesario pagar la api para usarla, pero en principio solo se va a usar en las primeras versiónes.
Y todo lo que se usara sera de código abierto, ademas que este proyecto no tiene animo de lucro por lo que no es necesario pagar licencias.

De ser un proyecto con animo de lucro habria que comprar la licencia de la api probablemente, o modificar los datos pasados de la api a la base de datos para no tener que pagar, total esos datos almacenados son publicos
