# Presupuesto

En un principio el proyecto no requerira ningun pago puesto que es de código abierto. Pero de comercializarse el mismo, tendriamos que tener en cuenta las horas totales de trabajo del programador unas 100 horas al finalizar el proyecto aproximandamente. El uso de la api que estoy usando para uso comercial es de aproximadamente 149€ al mes lo cual se evitaria pagar si es posible obteniendo los datos de otras formas. Habria que pagar las horas del programador poniendo que son 100 horas a 10€ la hora serian unos 1000€ por el trabajo realizado. Para monetizar el proyecto se incluirian anuncios no invasivos como pequeños anuncios en los laterales de la aplicación que no molesten. Pero como he dicho anteriormente esto solo es hipotetico porque este proyecto no esta destinado a un uso comercial.
