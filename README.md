# CustomGameDB

Esta aplicacion accede a una base datos de videojuegos. Actualmente esta en una primera versión y se accede a una api para obtener los datos y mostrarlos. Actualmente se puede buscar en la aplicacion con un buscador insertando un nombre, ver datos generales de los mismos y ver datos mas especificos al pulsar encima de cada juego. En siguentes versiones del proyecto la intención es volcar gran parte de los datos de la api en una base de datos y acceder desde ahi a los mismos, se podra añadir, modificar datos desde la misma, se implementara tambien un sistema de logeo para que los usuarios puedan crear una cuenta. Y tambien se implementara un chat entre distintos usuarios para que puedan hablar desde la aplicación.

Esta aplicación resuelve la necesidad de consultar información sobre videojuegos, guardar tus propias reseñas y compartirlas con otros usuarios. Su publico objetivo son personas que le gusten los videojuegos.

Para este proyecto se ha usado .NET con wpf, la api rawg para los datos con el objetivo de hacer una aplicación para optimizada para windows. El estado del proyecto es una fase bastante inicial del mismo donde aun quedan muchas cosas por implementar, aunque sus caracteristicas basicas ya se pueden usar.
