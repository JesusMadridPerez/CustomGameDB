# Prueba

Para probar este proyecto tendras que irte a la sección de releases para descargarte la ultima versión.
Sigue las indicaciones del instalador y una vez este completa la instalacion podras empezar a usarla.
En la aplicacion puedes navegar por la api buscando datos de los juegos que quieras buscar. Puedes buscar por
nombre del juego. Al clicar sobre un juego te saldra información mas detallada sobre el mismo.
